# Syntax

## Default Word Order

(VSO) Verb Subject Object

## Adjectives

* Adjectives are derived from Nouns
* Adjectives come after Nouns

## Ad-positions/Prepositions

* Ad-positions comes from Verbs
* Comes before the Nouns

## Possession

* Possesses come before Possessor
